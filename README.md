# To-Do List Web Application

A simple and interactive task management web application.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Data stored using Local Storage

## Technologies Used
- HTML
- CSS
- JavaScript

## Preview
Simple UI for managing daily tasks efficiently.

## How to Run
1. Download or clone the repository
2. Open `index.html` in your browser

## Future Improvements
- Add due dates
- Add task categories
- Improve UI design
